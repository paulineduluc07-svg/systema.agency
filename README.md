# Systema.agency

App de productivité tout-en-un pour neurodivergents.

## Technologies

- **Frontend:** React 19 + Tailwind CSS + shadcn/ui
- **Backend:** Node.js + Express + tRPC
- **Base de données:** MySQL (Drizzle ORM)
- **Authentification:** OAuth

## Installation locale

```bash
# Installer les dépendances
pnpm install

# Configurer les variables d'environnement
cp .env.example .env

# Lancer en mode développement
pnpm dev
```

## Fonctionnalités

- Dashboard avec widgets déplaçables
- Système d'onglets personnalisables
- Tâches et notes
- Mode sombre/clair
- Synchronisation cloud
- Export PDF
